Smart Agriculture Dashboard

Real-time IoT crop monitoring simulator built with HTML5, CSS3, Canvas API,JS.

Features
- Live crop monitoring (Paddy, Sugarcane, Cotton, Soyabean)
- Irrigation & fertilizer controls with instant feedback
- Sparkline charts & health analytics
- Dark mode, mobile-responsive
- localStorage persistence
- CSV export

#Tech Stack
- HTML5 Semantic
- CSS Grid/Flexbox + Custom Properties
- Canvas API for charts
- JavaScript
